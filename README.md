# Journalistieke Voetbalagent

## Doel
Dit project toont een agentic AI-toepassing die autonoom
de nieuwste voetbaluitslagen en actueel voetbalnieuws verzamelt,
selecteert en samenvat voor lezers.

## Journalistieke relevantie
Voetbaljournalisten werken dagelijks met grote hoeveelheden
actuele data. Deze agent automatiseert die routine en presenteert
de informatie in een overzichtelijk platform.

## Gebruikte techniek
Custom GPT ingericht als journalistieke agent.

## Werkwijze (agentic)
De agent werkt autonoom in meerdere stappen:
1. Verzamelen van recente voetbaluitslagen
2. Verzamelen van actueel voetbalnieuws
3. Selectie op relevantie en actualiteit
4. Samenvatten in neutrale journalistieke taal
5. Structureren van output als platformoverzicht

## Resultaat
Een tekstueel platform dat fungeert als mini-nieuwsoverzicht
met uitslagen en voetbalnieuws.
